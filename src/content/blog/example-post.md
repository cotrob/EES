# Welcome to Our Blog

This is an example blog post. Delete this file and create your own posts in the `src/content/blog/` directory.

## How to Create Posts

Create a new `.md` or `.mdx` file in `src/content/blog/` with frontmatter like this:

```yaml
---
title: "Your Post Title"
description: "A brief description for SEO"
pubDate: 2024-01-15
author: "Author Name"
image:
  src: "/images/post-image.jpg"
  alt: "Description of image"
tags: ["tag1", "tag2"]
draft: false
---
```

## Markdown Features
